Kuesioner Pengalaman Penggunaan Aplikasi Whoosh

Halo teman-teman!

Perkenalkan nama saya Aliyah Nisa Lathifah mahasiswi jurusan Informatika Universitas Gunadarma. Saat ini, saya sedang menjalani proses penyusunan penelitian ilmiah yang berjudul "Platform Berbasis Web untuk Menganalisis Pengalaman Pengguna pada Aplikasi Whoosh Menggunakan Metode User Experience Questionnaire (UEQ)" guna menyelesaikan salah satu syarat untuk mencapai gelar Strata Satu (S1).

Pengambilan data memerlukan waktu kurang lebih 3 menit.

Untuk itu saya mohon kesediaan Anda untuk mengisi kuesioner yang berisi beberapa pertanyaan singkat mengenai pengalaman Anda dalam menggunakan Aplikasi Whoosh.

Berikut sekilas tampilan Aplikasi Whoosh: https://bit.ly/TampilanAplikasiWhoosh
